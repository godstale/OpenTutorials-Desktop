# PennyPress 사이드바 미구현 메뉴 레이아웃 정리

## 작업 개요
사용자 포털 사이드바(`UserSidebar.tsx`)에서 아직 개발되지 않은 기능인 **호스팅 관리** 및 **결제 및 요금제** 메뉴에 대해 시각적 분리 및 비활성화(Disable) 처리를 진행한 작업 내용입니다.

## 수정 내용

1. **미구현 메뉴 비활성화 (Disable)**
   - 기존 `<Link>` 래핑을 제거하여 클릭 및 페이지 이동을 원천 차단했습니다.
   - `SidebarMenuButton` 컴포넌트에 `onClick={(e) => e.preventDefault()}`, `cursor-not-allowed`, `opacity-50` 속성 및 테일윈드 클래스를 적용하여 시각적/기능적으로 비활성화 상태임을 표시했습니다.
   - 사용자가 메뉴에 마우스를 올릴 때 (Hover) `"추후 제공될 예정입니다"` 라는 툴팁이 나타나도록 `tooltip` 속성을 추가했습니다. (네이티브 `disabled` 속성을 사용하면 pointer-events가 차단되어 툴팁 이벤트가 발생하지 않으므로, 커스텀 스타일로 우회하여 구현했습니다.)

2. **구분선(Separator) 추가 및 메뉴 그룹화**
   - `@/components/ui/sidebar`의 `SidebarSeparator`를 활용하여 사이드바 메뉴 간 시각적 구분선을 추가했습니다.
   - 다음과 같이 논리적 그룹을 지어 사용성을 향상했습니다:
     - **1그룹**: `대시보드`
     - _(구분선)_
     - **2그룹**: `에이전트 검색`, `외부 에이전트` (현재 사용 가능한 핵심 기능)
     - _(구분선)_
     - **3그룹**: `호스팅 관리`, `결제 및 요금제` (미구현 기능)
     - _(구분선)_
     - **4그룹**: `설정`

## 주요 변경 파일
- `components/layout/UserSidebar.tsx`: 메뉴 렌더링 매핑 로직을 명시적 하드코딩 구조로 변경하여 각 그룹과 툴팁을 세밀하게 제어하도록 수정.
