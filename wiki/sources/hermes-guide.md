# Hermes Agent 완전 가이드

## Summary
Hermes Agent(v0.14.0)의 5가지 핵심 시스템, 설치 방법, 인증, 설정, 도구, 스킬 시스템, 지속 메모리, 메시징 게이트웨이 및 멀티 에이전트 칸반 등을 망라한 종합 레퍼런스 가이드.

## Key Concepts
- [[HermesArchitecture]]: 5가지 핵심 시스템 및 설정/인증 구조
- [[HermesSkills]]: 온디맨드 지식 문서 기반 스킬 시스템
- [[HermesCron]]: 에이전트 작업 단위 예약 및 MCP 통합
- [[HermesSecurity]]: Secret redaction, SSRF 차단, 샌드박싱 등 보안 기능
