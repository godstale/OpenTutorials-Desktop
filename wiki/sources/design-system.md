# PennyPress Design System & UI Guide

- **Source**: `DESIGN.md`
- **Summary**: PennyPress FE의 공통 디자인 시스템 및 UI/UX 가이드. Tailwind CSS 색상 팔레트, 타이포그래피, 레이아웃 구조(User/Admin), Shadcn UI 컴포넌트 사용 규칙 및 상호작용 패턴을 정의.

## Key Points
- **Color**: Zinc/Slate 기반의 현대적이고 전문적인 테마. Primary는 대비되는 흑/백, Destructive/Success 등 포인트 컬러 지정.
- **Layout**: User Portal(Header + 좌측 Sidebar, 1024px 최소 폭), Admin Portal(어두운 헤더 + 밝은 사이드바)
- **Components**: Card, Button, Badge, Table, Dialog(Modal), Avatar 위주로 사용하며, Hover Effect와 일관된 빈 상태(Empty State) 디자인 유지.
